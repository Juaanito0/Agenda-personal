# 📒 Agenda Personal

Este proyecto es una **agenda digital** desarrollada en Python que permite gestionar contactos y organizar información de manera sencilla.  
Forma parte de mi proceso de aprendizaje en desarrollo web y backend, integrando conceptos de bases de datos y lógica de programación.

## 🚀 Funcionalidades
- Registro de usuarios con formulario de inicio de sesión.
- Gestión de contactos (crear, editar, eliminar).
- Almacenamiento en base de datos SQLite.
- Interfaz básica en consola.

## 🛠️ Tecnologías utilizadas
- **Python 3.13**
- **SQLite** para la base de datos
- Manejo de módulos y archivos `.py`

## 📂 Estructura del proyecto
- `agenda.py` → Lógica principal de la agenda.
- `db.py` → Conexión y operaciones con la base de datos.
- `formulario_login.py` → Módulo para autenticación de usuarios.
- `.gitignore` → Configuración para excluir archivos innecesarios.
